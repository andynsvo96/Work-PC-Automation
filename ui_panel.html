Auto Splitter

Read More

Auto Splitter first scans the original CRM order, checks the expected tab count, and builds the split plan. If that dry run preflight passes, the live split creates the split orders and updates the original order.

If the dry run preflight fails, the live split is skipped and the error explains why.
